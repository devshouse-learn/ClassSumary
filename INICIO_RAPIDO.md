# 🚀 Guía de Inicio Rápido - ClassSummary

## ⚡ Instalación Express (2 minutos)

### Paso 1: Instalar Dependencias
```bash
cd ClassSummary
npm install
```

### Paso 2: Iniciar la Aplicación
```bash
npm start
```

### Paso 3: Abrir en el Navegador
La aplicación se abrirá automáticamente en:
```
http://localhost:3000
```

---

## 👤 Primer Uso

### Opción 1: Modo Invitado (Más Rápido)
1. Haz clic en **"Continuar como Invitado"**
2. ¡Listo! Ya puedes usar la aplicación

### Opción 2: Crear Cuenta
1. Haz clic en **"Registrarse"**
2. Ingresa:
   - Nombre completo
   - Email
   - Contraseña (mínimo 6 caracteres)
3. Haz clic en **"Registrarse"**

### Opción 3: Iniciar Sesión
Si ya tienes cuenta:
1. Haz clic en **"Iniciar Sesión"**
2. Ingresa email y contraseña
3. Haz clic en **"Iniciar Sesión"**

---

## 🎙️ Procesar Tu Primera Clase

### 1. Ir a Subir Audio
- Haz clic en **📤 Subir Audio** en el menú lateral

### 2. Seleccionar Archivo
- **Opción A**: Arrastra tu archivo de audio
- **Opción B**: Haz clic y selecciona desde tu computadora

### 3. Procesar
- Haz clic en **"Procesar con IA"**
- Espera 2-5 minutos (verás una barra de progreso)

### 4. Ver Resultados
Obtendrás:
- ✅ Resumen completo
- ✅ Puntos clave
- ✅ Preguntas de repaso
- ✅ Transcripción completa

---

## 🌍 Cambiar Idioma

1. Ve a **⚙️ Configuración** en el menú
2. Haz clic en el selector de idioma
3. Elige entre 30+ idiomas disponibles
4. ¡Listo! La interfaz se actualiza automáticamente

---

## 💾 Gestionar Tus Clases

### Ver Historial
1. Haz clic en **📚 Historial** en el menú
2. Verás todas tus clases procesadas
3. Usa el buscador para filtrar

### Descargar Resumen
1. Abre cualquier clase del historial
2. Haz clic en **💾 Descargar PDF**

### Copiar Resumen
1. Haz clic en **📋 Copiar**
2. El resumen se copia al portapapeles

---

## ⚙️ Configuración Básica

### Preferencias
- **Guardar automáticamente**: Activa para guardar todas las clases
- **Notificaciones**: Activa para recibir alertas

### Exportar Datos
1. Ve a **Configuración**
2. Haz clic en **"Exportar Datos"**
3. Se descarga un archivo JSON con toda tu información

---

## ❓ Ayuda Rápida

### Problemas Comunes

**No se carga el archivo**
- Verifica que sea un formato de audio válido (MP3, WAV, M4A, OGG)
- El tamaño máximo es 500 MB

**El procesamiento es lento**
- Es normal, puede tomar 2-5 minutos
- Depende de la duración del audio

**Perdí mi sesión**
- Si eres invitado, los datos son temporales
- Crea una cuenta para guardar permanentemente

## 🔒 Seguridad

### Datos Locales
- Tus archivos NO se envían a servidores externos
- Todo se procesa y guarda en tu navegador
- Privacidad 100% garantizada

### Modo Invitado
- Datos temporales en localStorage
- Se borran al cerrar el navegador
- Recomendamos crear cuenta para guardar permanentemente
