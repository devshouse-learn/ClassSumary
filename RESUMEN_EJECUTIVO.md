# 📊 Resumen Ejecutivo - ClassSummary

## 🚀 Cómo Ejecutar

### Instalación (Primera vez)
```bash
cd ClassSummary
npm install
```

### Iniciar Aplicación
```bash
npm start
```

### Build para Producción
```bash
npm run build
```

La aplicación se abrirá en: `http://localhost:3000`
